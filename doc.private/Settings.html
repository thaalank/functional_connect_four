Module Settings

module Settings: sig .. end

color are the different colors pieces and the background can be colored

selection is either the yellow pieces, the red pieces, or the background for color changing.

type color =
| Black
| White
| Red
| Green
| Blue
| Yellow
| Cyan
| Magenta 
val yellow_color : color Stdlib.ref

yellow_color is a reference to the color of the yellow pieces. (This sounds dumb, but it makes sense when you consider that the State module defines pieces based on Yellow/Red)

val red_color : color Stdlib.ref

red_color is a reference to the color of the red pieces. (This sounds dumb, but it makes sense when you consider that the State module defines pieces based on Yellow/Red)

val background_color : color Stdlib.ref

background_color is a reference to the color of the background

val next_color : color -> color

next_color color is the next color in the sequence of colors

val prev_color : color -> color

prev_color color is the previous color in the sequence of colors

val to_color : color -> Graphics.color

to_color color is the Graphics color of color

val to_alt_color : color -> Graphics.color

to_alt_color color is the Graphics color of color, except slightly differently colored.

val change_setting : ('a -> 'a) -> 'a Stdlib.ref -> unit

change_setting f s applies f to setting s

val set_setting : 'a -> 'a Stdlib.ref -> unit

change_setting c s sets setting s to value c

val circ : Window.region ->
       color -> ?offset:float * float -> ?alt:bool -> float -> Screen.shape

circ ~offset:(x_off, y_off) ~alt:alter factor r c is a circle at (r.x, r.y) offset by (x_off, y_off) relative to 40% of the region's width and height. The size of the circle is scaled by factor. The circle has an alternative color to c if alt is true.

type selection =
| YellowSelection
| RedSelection
| Background 
val to_ref : selection -> color Stdlib.ref

to_ref sel is the reference sel corresponds to

val advance_color : selection -> unit

advance_color sel advances the color of the selection to the next color.

val devance_color : selection -> unit

devance_color sel retreats the color of the selection to the previous color.

val reset_color : selection -> unit

reset_color sel resets the color of the selection. red for red pieces; yellow for yellow pieces; blue for the background

val get_color : selection -> Graphics.color

get_color sel gets the Graphics color of the selection

val get_alt_color : selection -> Graphics.color

get_alt_color sel gets the alternative Graphics color of the selection

type piece =
| YellowPiece
| RedPiece

piece is a choice of the red or yellow pieces

val to_ref' : piece -> style Stdlib.ref

to_ref' pi is the reference corresponding to piece pi

val advance_style : piece -> unit

advance_style pi advances the style of pi to the next style

val devance_style : piece -> unit

devance_color sel retreats the style of pi to the previous style

val reset_style : piece -> unit

reset_style pi resets the style of pi to CIRCLE

val get_shape : piece -> Window.region -> Screen.shape

get_shape pi r is the Screen shape corresponding to pi, with the style and color of pi, within r

val get_desc : piece -> string

get_desc pi is a description of the style of pi
