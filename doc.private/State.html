The representation type of a token.

type token =
| Yellow
| Red
| Empty 
type col = token list

The abstract representation type of a column.

type turn =
| YellowTurn
| RedTurn
| RedWins
| YellowWins
| Tie

The representation type of whose turn it is or if the game is over.

type t = col list * turn

The abstract representation type of a state in the game.

exception FullCol

Raised when when a column is full.

exception InvalidCol

Raised when an index is not a valid column index.

exception InvalidRow

Raised when an index is not a valid row index.

val make_empty_col : int -> token list

make_empty_col r is an Empty token list with r Empty tokens, which represents one empty column.

val make_empty_grid : int -> int -> token list list

make_empty_grid r c is an Empty grid with r rows and c columns.

val make_start_state : int -> int -> token list list * turn

make_start_state r c is the State.t with a board of r rows, c columns, and with YellowTurn as the first turn .

val get_col : int -> 'a list -> 'a

get_col c_i cols is the column at index c_i in cols. Raises: InvalidCol if c_i is not a valid column index.

val get_token_row : int -> 'a list -> 'a

get_token_row r_i col is the token at index r_i in col. Raises: InvalidRow if r_i is not a valid row index.

val get_token : int -> int -> token list list * 'a -> token

get_token c_i r_i t is the token in t at column c_i and row r_i. Raises: InvalidCol if c_i is not a valid column index. Raises: InvalidRow if r_i is not a valid row index.

val get_turn : 'a * turn -> turn

get_turn t is the turn at state t.

val col_to_list_helper : token list -> col -> token list

col_to_list_helper acc col is a token list representation of col but reversed.

val col_to_list : token list -> col -> token list

col_to_list acc col is a token list representation of col.

val grid_to_list_helper : token list list -> col list -> token list list

grid_to_list_helper acc grid is a token list list representation of grid but reversed.

val grid_to_list : token list list -> col list -> token list list

grid_to_list acc grid is a token list list representation of grid.

val get_board : t -> token list list

get_board t is the board at state t represented as a list of token lists.

val consec4 : 'a list -> 'a -> bool

consec4 tlist token is whether the tlist has four consecutive token.

val vert4 : col list -> token -> bool

vert4 lst token is whether token satisfies the vertical winning condition in the given lst orientation.

val get_idx : 'a list -> int -> int -> 'a

get_idx col r_i count gets the desired index of the col by matching r_i and count.

val hori4 : col list -> token -> bool

hori4 lst token is whether token satisfies the horizontal winning condition in the given lst orientation.

val square4x4 : col list -> token -> bool

square4x4 lst token is whether token satisfies the winning condition in the given lst orientation.

val square_larger : col list -> token list list

square larger lst is a helper for diag4. Fails with "Error : invalid game board", if the lst is empty.

val is_col_full : token list -> bool

is_col_full col is true if the column col is full. false if not.

val check_rect : col list -> col list

check_rect lst is a helper for diag4.

val diag4 : col list -> token -> bool

diag4 lst token is whether the player using token in lst has met the diagonal winning condition.

val check_win_condition : col list -> token -> bool

win_condition lst token is true if the current game lst is a win for one of the two players using token and false otherwise.

val is_col_full : token list -> bool

is_col_full col is true if the column col is full. false if not.

val drop_in_col : token list -> token -> 'a -> col -> col

drop_in_col col token c_i acc is col with token dropped into it.

val new_col : token -> int -> token list list -> token list

new_col token c_i grid is the new column at c_i in grid with token dropped into it.

val drop_token : token ->
       int -> col list -> col list -> int -> col list

drop_token token c_i grid acc n is the new grid with token dropped at c_i.

val next_turn : turn -> turn

next_turn turn is the next turn if the current turn is turn. Fails with "some winning condition/no new turn game is over." if there is no next turn.

val is_grid_full : token list list -> bool

is_grid_full lst is true if the grid represented by lst is full of tokens. Otherwise, false.

val get_winner : 'a -> token -> 'a * turn

get_winner grid token is the state associated with the proper winning turn based on token.

val drop : token ->
       int -> col list * turn -> col list * turn

drop token c_i t is the state after dropping token at col_i in state t. Fails with "not a valid token" if the winning conditions are met but token is not a valid token to meet the winning conditions.

val get_possible_moves_helper : token list list -> int list -> int -> int list

get_possible_moves_helper board acc n is a list of all the possible col_i that a token can legally be dropped into board.

val get_possible_moves : token list list * 'a -> int list

get_possible_moves t is a list of all the possible col_i that a token can legally be dropped into in t.
